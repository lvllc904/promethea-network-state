The Architecture of Foresight

Traditional trading bots react to the past. The Promethea framework imagines the future. This infographic visualizes its core components, from its multi-layered perception of the market to its generative decision-making process.

From Reaction to Generative Foresight

1. Perceive Reality

The agent ingests a high-dimensional Unified State Vector, capturing a rich, multi-layered snapshot of the market ecosystem.

→

2. Imagine Futures

A Generative World Model simulates thousands of plausible future market trajectories based on potential actions.

→

3. Plan & Act

A Planning Algorithm (MCTS) explores these simulated futures to select the action with the highest expected long-term utility.

The Unified State Vector (Sₜ)

The agent's perception of reality is a composite of six distinct vectors, each capturing a different layer of the market. This is the sole input for its World Model.

📈Core Technicals

The ground truth of price, volume, and standard derived indicators. This vector provides the fundamental quantitative basis for the agent's analysis.

A line chart showing price action and key moving averages, representing a fraction of the technical data ingested by the agent.

🌍Economic Vector

Anchors the agent in fundamental reality by tracking key macroeconomic indicators like interest rates, inflation (CPI), and volatility (VIX).

A radar chart visualizing the current state of key macroeconomic indicators, normalized to show their relative strength or weakness.

👥Psychology Vector

Quantifies the market's collective mindset by tracking sentiment distribution, shifts in public attention, and the influence of new AI-driven narratives.

A donut chart representing the sentiment distribution (δt) across public discourse, a key measure of market consensus.

💥Chaos Vector

Measures the market's internal stability and its propensity for sudden, chaotic state changes. It acts as an early warning system for events like flash crashes by tracking volatility acceleration and the breakdown of historical correlations.

🔗Entanglement Vector

Models the interconnectedness between assets. It measures how price, volatility, and sentiment shocks propagate through the market network, revealing hidden relationships and contagion risks between different parts of the financial ecosystem.

🧠Reflexivity Vector

Grants the agent self-awareness by modeling the market's feedback loops. It tracks the popularity of trading narratives (Dt), trade crowdedness (Ct), and the agent's own correlation to common strategies (ρM(t)) to avoid being exploited.

The Planning Algorithm: MCTS

Instead of a reactive policy, the agent uses Monte Carlo Tree Search (MCTS) to "imagine" thousands of futures and plan its optimal action.

Selection

Traverse the existing tree of simulated futures, prioritizing promising paths.

→

Expansion

When an unexplored path is reached, use the World Model to create a new future state.

→

Simulation

Run a fast, lightweight simulation (rollout) from the new state to estimate its value.

→

Backpropagation

Update the value of all parent nodes based on the simulation's outcome.

Validation: The Adversarial Backtest

A standard backtest is insufficient. To ensure true resilience, the Promethea agent is tested against an adversary designed to exploit its strategy.

Generative Agent (G)

Executes its strategy to maximize risk-adjusted returns.

⚔️

Adapts & Resists

Adversarial Agent (A)

Actively creates market conditions to cause G to lose money.

The Generative Agent is considered robust only if it maintains profitability in this hostile, adaptive environment.
